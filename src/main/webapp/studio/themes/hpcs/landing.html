Installing FHC

The FHC command line utility gives access to the entire FeedHenry suite through your command line. It's a NodeJS utility which allows you to quickly get started building your apps in HP Cloud Services.

$ sudo npm install -g fh-fhc 
$ fhc -v
$ fhc target hpcs.feedhenry.com
$ fhc login email password
